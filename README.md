#  Assignment 1 – React Multi-Page Website


**Assignment 1** is a React-based web project built using **React Router DOM** and **Tailwind CSS**.  
The application includes a clean layout with reusable components like a **Navbar**, **Footer**, and multiple pages — **Home**, **Projects**, **Contact**, and a **404 Not Found** page.

This project demonstrates understanding of:
- React component structure  
- Routing and navigation  
- Layout handling using `<Outlet />`  
- Tailwind CSS for styling  

---

## Features
- **Home Page:** Simple introduction section  
- **About Page:** Details about internship projects (CeipaLabs)  
- **Contact Page:** Contact details and links  
- **404 Page:** Custom “Page Not Found” screen for invalid URLs  
- **Layout:** Shared Navbar and Footer across all pages  
-  **Tailwind CSS:** For responsive, clean, and fast styling  

---

##  Tech Stack
- **React.js**  
- **React Router DOM (v6)**  
- **Tailwind CSS**  
- **Vite** or **Create React App**
